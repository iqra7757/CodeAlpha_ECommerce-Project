# E-Commerce Project - @CodeAlpha Internship Task.

A complete online shopping solution with product management and user authentication.

## 🚀 Key Features:
- **Product Catalog:** View products by categories.
- **Shopping Cart:** Add, remove, and update quantities.
- **User Authentication:** Secure login and registration.
- **Admin Dashboard:** Manage products and orders.

## 🛠️ Tech Stack:
- ASP.NET Core MVC
- Entity Framework Core (SQL Server)
- Bootstrap & JavaScript
